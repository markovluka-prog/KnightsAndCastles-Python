# Рыцари и Замки / Knights and Castles

![Knights and Castles](Knights_and_Castles_1920x1080.png)

Тактическая пошаговая стратегия на двоих или против ИИ.

## Запуск

```bash
pip install pygame
python3 knights_and_castles.py
```

## Управление

| Действие | Клавиша |
|---|---|
| Выбрать / переместить юнит | ЛКМ |
| Просмотр фигурки (увеличенно) | ЛКМ по юниту |
| Пропустить юнит | ПКМ / Space |
| Вернуться в меню / Выход | ESC |

## Режимы

- **2 Игрока** — локальная игра на одном экране
- **Против ИИ** — игра против тактического ИИ-противника

## Юниты

| Юнит | HP | Урон | Броня | Ходы |
|---|---|---|---|---|
| Рыцарь | 3 | 5 | 4 | 2 |
| Конный рыцарь | 5 | 6 | 3 | 3 |
| Лучник | 3 | 2 | 1 | 3 |
